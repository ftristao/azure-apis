# azure-apis
Servless apis para rodar no Azure (DIO)

# Testes
Para executar, inicialize o mongodb (via docker: script runDocker.sh).
E inicialize a base com o arquivo ./shared/init.json

Rode a aplicação com:
npm install
function host start
